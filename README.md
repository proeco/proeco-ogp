# proeco-ogp

## description
query に
```
{
  title: string;
  teamName: string;
  teamIconUrl: string;
}
```
を渡すことでOGP画像を生成します

## example
```
https://proeco-ogp.vercel.app/api/ogp?title=hogehuga&teamName=penguin&teamIconUrl=https://itizawa-tech.growi.cloud/attachment/619cf4648062e8c5ac7e684f
```
